# 📈 Institutional Buy & Sell [SMC + Wyckoff]

> TradingView PineScript v5 indicator — Smart Money Concept + Wyckoff + Multi-Indicator Dashboard

---

## 🧠 핵심 철학

**"좋은 매수는 저점에서만 나온다"**

일반적인 스크립트들은 상승 추세면 어디서든 BUY를 발사합니다.  
이 스크립트는 **2단계 Gate 시스템**으로 매집 구간에서만 BUY가 발동됩니다.

```
GATE 1: 최근 30봉 범위의 하위 30% 이내 (가격 위치 필터)
GATE 2: RSI < 50 (아직 과매수가 아닌 상태)

두 GATE 모두 통과 + ACCUM Score ≥ 45 → ▲ BUY 발동
```

## 📊 우상단 대시보드 설명

차트 우상단에 실시간으로 업데이트되는 14개 항목의 투자 대시보드입니다.

| INDICATOR | 설명 | 해석 방법 |
|-----------|------|-----------|
| **Trend** | EMA 50/200 기반 추세 방향 | 🟢 BULL = 상승추세 / 🔴 BEAR = 하락추세 |
| **VWAP** | 기관 평균 매수가 vs 현재가 | Above = 기관보다 비싸게 거래 중 / **Below = 기관 평균보다 싸게 살 수 있음 (매수 유리)** |
| **RSI** | 상대강도지수 (14) | < 30 → OVERSOLD 🟢 (매수 존) / > 70 → OVERBOUGHT 🔴 (매도 존) |
| **RVOL** | 상대 거래량 (현재 거래량 ÷ 20일 평균) | 1.5x+ = 세력 진입 🔵 / 0.6x 미만 = 매도세 소진 (주황) |
| **Stoch** | 스토캐스틱 K/D | OVERSOLD 🟢 = 과매도 반전 가능 / OVERBOUGHT 🔴 = 고점 경고 |
| **MACD** | 모멘텀 방향 | CROSS UP ▲ = 반전 시작 🟢 / BEARISH = 하락 모멘텀 |
| **ADX** | 추세 강도 | > 25 → STRONG (추세 신뢰) 🔵 / < 20 → WEAK (횡보, 신호 주의) |
| **ATR %** | 변동성 (ATR ÷ 현재가 × 100) | > 4% = 고변동 🔴 (포지션 사이즈 줄일 것) / < 2% = 저변동 |
| **BB Sqz** | 볼린저 밴드 스퀴즈 감지 | ⚡ SQUEEZE = 변동성 극도로 수축, 곧 폭발 임박 |
| **52 Week** | 52주 고저 대비 현재 위치 (%) | LOW (< 25%) 🟢 = 연간 저점 근처 / HIGH (> 75%) 🔴 = 연간 고점 근처 |
| **Position** | 최근 30봉 기준 현재 가격 위치 | LOW Zone (< 25%) = 매집 적합 구간 🟢 |
| **ACCUM** | 매집 종합 점수 | 숫자 / 45 → ▲ BUY 발동 시 배경 🟢로 변경 |
| **SELL** | 매도 종합 점수 | 숫자 / 50 → ▼ SELL 발동 시 배경 🔴로 변경 |

> 아래 8가지 조건이 많이 겹칠수록 최적의 매수 타이밍입니다.

| # | 조건 | 의미 | 충족 시 |
|---|------|------|---------|
| ✅ 1 | **VWAP Below** | 기관 평균 매수가보다 현재가가 낮음 → 세력보다 싸게 살 수 있음 | 🟢 |
| ✅ 2 | **RSI < 40** | 과매도 영역 진입 → 이미 많이 빠진 상태 | 🟢 |
| ✅ 3 | **BB Squeeze** | 볼린저 밴드 수축 → 큰 움직임이 곧 시작될 것 | 🟢 |
| ✅ 4 | **LOW Zone 25% 이내** | 최근 30봉 가격 범위의 하위 25% 구간 → 단기 바닥권 | 🟢 |
| ✅ 5 | **52W Low 30% 이내** | 년봉 기준 저점 근처 → 연간 최저 근처에서 매수 | 🟢 |
| ✅ 6 | **Stoch Oversold** | 스토캐스틱 K/D 모두 20 미만 → 단기 반전 준비 | 🟢 |
| ✅ 7 | **Bullish Divergence** | 가격은 저점 갱신, RSI는 저점 높아짐 → 매도세 약화 | 🟢 |
| ✅ 8 | **거래량 소멸 / 매도 클라이맥스** | 매도세 고갈 또는 대량 매도 후 흡수 → 세력 매집 가능성 | 🟢 |

### 📊 조건 충족 숫자별 해석

| 충족 수 | 헤더 표시 | 해석 |
|---------|-----------|------|
| **6 ~ 8개** | 🏆 `N / 8  PRIME ZONE` (금색) | **최고의 매수 타이밍. 강하게 고려할 것** |
| **4 ~ 5개** | ⭐ `N / 8  Watch Zone` (청록) | 가능성 있음. 추가 확인 후 진입 |
| **0 ~ 3개** | `N / 8  Not Ready` (회색) | 아직 조건 미충족. 관망 권장 |

> 💡 **팁**: PRIME ZONE + `▲ BUY` 화살표가 동시에 뜰 때가 가장 강력한 신호입니다.
